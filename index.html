Spotify Streams

Project Description

This project features an interactive dashboard that analyzes Spotify song data, created to demonstrate my data visualization and analytical skills using Power BI. The dashboard provides deep insights into various aspects of Spotify songs, including popularity trends, genre analysis, and audio feature distributions.

One of the focal points of this analysis is the creation of a CashFlow statement.

Key Features:

  1. Data Overview:
    • An overview of the Spotify dataset, including key metrics such as the number of songs releases and streams.
    • A summary of the number of released songs across the years and the most popular songs
  2. Audio Feature Analysis:
    • Analysis of audio features such as danceability, energy, tempo, and valence.
    • Radar plots and data cards to illustrate the distribution of these audio features across songs.
  3. Interactive Dashboards:
    • Interactive dashboards that allow users to explore data by artist, release date, and album.
    • Implementation of filters and slicers to enable customized views and deeper analysis of specific data segments.

Technologies Used:

  • SQL: For data cleaning and preparation.
  • Power BI: For data visualization and dashboard creation.
  • DAX (Data Analysis Expressions): For creating calculated columns and measures to enhance data analysis.
  • Power Query: For data transformation.

Eurovision Song Contest Analysis using Power BI

Project Description

This project presents an analytical report on Eurovision Song Contest data, created to participate in a company contest. The report aims to answer specific questions related to the Eurovision Song Contest, providing deep insights into various aspects of the competition since 2009. One of the main features og this report is the navigation panel that appears once you open the report. This navigation panel helps the user understand how to navigate the report and how to extract the desired insights.

Key Features:

  1. Data Overview:
    • An overview of Eurovision data, including participants, songs, and scores since 2009.
    • Key metrics such as total points, average points per style, and the main characteristics of each song.
  2. Songs Analysis:
    • Detailed analysis of the main song characteristics for each song, style and country
    • Visualization tools such as bar charts, line graphs, maps and scatter plots are used to present the average points.
  3. Key Insights and Findings:
    • Summary of key insights derived from the data analysis, such as the most successful song types and performance characteristics.
    • Highlighting interesting patterns and anomalies in Eurovision voting and performance trends.

Technologies Used:

  • Power BI: For data visualization and dashboard creation.
  • DAX (Data Analysis Expressions): For creating calculated columns and measures to enhance data analysis.
  • Power Query: For data transformation and preparation.
  • PowerPoint: For some of the visuals like the background and lettering
